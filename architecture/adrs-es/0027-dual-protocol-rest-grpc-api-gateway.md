## Resumen de Adaptación

The corporate standard is adopted with the following modificaciones específicas del proyecto:
1. UMS usa .NET gRPC (Grpc.AspNetCore). Mismo Kong Gateway, mismos contratos .proto.
2. Diferido: servicios gRPC internos hasta Fase 2.
