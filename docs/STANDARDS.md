> Language: [English](./STANDARDS.md) | [Español](./STANDARDS.es.md)

UMS is an applied implementation of Evolith. Reusable enterprise standards live in Evolith; UMS keeps product-specific evidence, local decisions, and approved deviations through ADRs.

## Authority rule

| Type | Source of truth |
|---|---|
| Reusable enterprise standard | Evolith |
| Product-specific implementation evidence | UMS |
| Local route, module, header, schema, seed or runtime value | UMS |
| Practice intended for all products | Promote to Evolith through ADR, standard or canonical pattern |
